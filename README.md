# 学习提醒~
学完一个语句后就复制例子到新程序里运行一遍就懂了<br>
出错了就看看下面的error提示，双击某一句错误提示就能跳到出错的地方<br>
多用用百度，多问问别人，多试几次bug就掌握了<br>
